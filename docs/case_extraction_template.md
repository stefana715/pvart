# Case Extraction Template

Use one row per project / installation.

## Basic identification
- case_id
- project_name
- designer / artist / architect
- year
- country / city
- source link / citation

## Typology
- integration_type (roof / facade / canopy / pavilion / landscape / public artwork / urban furniture / other)
- setting (building / campus / public square / park / waterfront / temporary exhibition / other)
- permanence (temporary / semi-permanent / permanent)
- scale (object / pavilion / building element / building / urban installation)

## PV and material information
- pv_type
- coloured_or_standard
- transparency_level
- custom_pattern_or_image (yes/no)
- installed_capacity_if_available
- energy_output_if_available

## Design and art dimensions
- primary_aesthetic_strategy
- artistic_autonomy_level (low / medium / high)
- public_interaction_level (low / medium / high)
- narrative_or_symbolic_content
- relation_to_context

## Practical dimensions
- stated_purpose
- stakeholders
- maintenance_notes
- known_constraints
- replicability_notes

## Notes
- short analytical summary
- image availability
- missing information
