# Project Brief

## Topic
PV × Art / aesthetic photovoltaics / public solar installations / artistic BIPV.

## Positioning
This project studies how photovoltaic systems can function not only as energy-generating devices, but also as:
- architectural materials,
- aesthetic surfaces,
- symbolic urban elements,
- and public-facing artistic media.

The project therefore sits between energy engineering, architecture, urban design, material aesthetics, and public art.

## Core academic opportunity
A conventional BIPV paper usually focuses on:
- energy yield,
- thermal performance,
- cost,
- lifecycle assessment,
- or technical integration.

This project can contribute by addressing the underdeveloped side of PV adoption:
- visual acceptance,
- artistic expression,
- urban identity,
- experiential value,
- and design translation into real projects.

## Main risk
If the paper only collects visually interesting examples, it will be too descriptive.
The publishable version must provide a framework, taxonomy, or assessment method.

## Preferred strategy
Start from a review + framework paper.
Later, if possible, upgrade with empirical evidence such as expert evaluation or user perception survey.

## Candidate titles
1. Photovoltaics as Architectural and Artistic Media: A review of design strategies for energy-aesthetic integration
2. From BIPV to Public Solar Art: Towards a multidimensional framework for evaluating photovoltaic integration beyond efficiency
3. Energy, Aesthetics, and Urban Expression: A review and assessment framework for PV-art applications

## Initial research tasks
1. extract the cases from current PPT/PDF/image materials
2. classify them by typology
3. collect related academic references
4. identify a gap in existing review literature
5. define a reproducible framework
